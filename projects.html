Monte Carlo Pathtracer

I implemented a Monte Carlo Pathtracer with various illumination techniques, including direct lighting and global illumination using multiple importance sampling. Features included refractive materials, Fresnel reflection, microfacet materials, and different types of light sources. This project took about 2 months and was written in C++ using Qt Creator. Raytracer

To prepare for writing the pathtracer, I implemented a raytracer and learned fundamentals including raycasting, lights, and materials that represent some of the various surface reflection and transmission functions. The raytracer also included sampling functions to implement effects like soft shadows and ambient occlusion.

Lego Construction

I built this web-based program to tackle "The Lego Problem"- Given any 3D body, how can we build it from Lego bricks? This program generates Lego brick models given an input mesh and parameters, and it creates an interactive environment to build virtual Legos. The goal was to create a fast process for designing and visualizing Lego structures by designing a building algorithm. I used JavaScript (three.js) and WebGL for the scripting and Maya to build each of the Lego brick models.
